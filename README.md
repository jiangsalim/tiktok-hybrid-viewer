# TikTok Hybrid Viewer

A TikTok video viewer that works without an account. Watch, search, and discover TikTok content with a personalized feed.

## Tech Stack
- Mobile: Kotlin + Jetpack Compose
- Backend: Go (gateway, peer, telemetry) + Python (trending) + Java (proxy)
- Native: C (JNI) + C++ (anti-tamper) + Rust (signatures)
- Database: Supabase (PostgreSQL)
